# init-jekyll

> Initial files for creating a Jekyll based site.

## Install

`npm i && bower i` to pull dependencies

# Usage

run `gulp` for development mode or `gulp --production` to build optimised version

## Features
- browser-sync
- css autoprefixing
- live reload
- etc.
