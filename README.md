# init-jekyll

Initial files for creating a jekyll site.

## Install

`npm i && bower i` to pull dependencies

# Usage

run `gulp` for development mode or `gulp --live` to build optimised version

## Features
- browser-sync
- uncss
- css autoprefixing
- live reload
- etc.
